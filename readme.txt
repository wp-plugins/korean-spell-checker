== Description ==

워드프레스 리치 에디터에 한글 맞춤법 검사기로 바로 가는 버튼을 삽입합니다. 이제 쪽팔리게 맞춤법 틀리지 마세요. (저도 못하지만)

한글 맞춤법 검사기는 따로 API가 없어서 링크로 대신해요.

공식 페이지 : <a href="http://www.sujinc.com/lab/korean-spell-checker/">한글 맞춤법 검사기!</a>

== Installation ==

1. 플러그인을 다운로드하여 압축을 해제해주세요

1. 플러그인 디렉토리에 넣어주세요

1. 플러그인 관리 페이지에서 활성화 시키세요
